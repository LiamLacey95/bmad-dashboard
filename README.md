# BMAD Dashboard

Web-based dashboard for monitoring and visualizing BMAD framework executions.

Built automatically by the BMAD framework itself.

## Overview

This dashboard provides real-time visibility into multi-agent workflows, cost tracking, and project progress for the BMAD framework.

## Features

- Real-time workflow monitoring
- Project management with Kanban board
- Cost tracking and agent analytics
- Document viewer (PRD, Architecture)
- WebSocket-based live updates
- Dark mode support

## Tech Stack

- Frontend: React 18 + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Build Tool: Vite

## Quick Start

```bash
npm install
npm run dev:server  # Port 3001
npm run dev:client  # Port 5173
```

Then open http://localhost:5173
